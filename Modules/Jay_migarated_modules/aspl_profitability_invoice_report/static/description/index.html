Odoo 17 Profitability By Invoice Report (Community)

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Generate Profitability By Invoice in PDF.
  • You can view report using group by Sales Team.

Available for Enterprise

How to use

  • Go to the Invoicing -> Custom Reports -> Profitability By Invoice Report.

  • Select Date Range to get report accordingly.
  • Select Customers which have product category and sales team.
  • Select Products to get Category.
  • Select Sales Team to get invoices for that sales team.
  • Select Report View to print report in summary view and datail view.
  • Select Show Credit to get reverse invoice record data or credit notes data.

  • Summary Report View : Profitability By Invoice Report


  • Detail Report View : Profitability By Invoice Report
