Odoo 17 Partner Address Label Report Extension (Community)

Partner address label design. User can create custom page label template from front end and can print the dynamic partner address label report.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • User has to select partner and no. of quantity to print partner label(s). User can create custom page label template format and can save design template for future use.
  • User need to select partner(s) and no. of quantity to print label(s).


  • After selecting partner(s),select the "Page" tab and fill the appropriate data for report page formatting.


  • If need to create custom paper type then select "Custom" as paper type and add page height and width as per requirement.


  • In barcode tab,fill appropriate data.


  • In design tab, user can change design of template and add, remove fields of partner in label design.


  • if need to save template design,than click on "Save Design" button. One prompt will open to save template design.


  • Print of dynamic partner address label.


  • Here we can see all saved template design.


  • User can also print partner label from 'Customers', 'Vendors', 'Sale Order', 'Account Invoice', 'Purchase Order', 'Stock Picking'.










  • In "Barcode Print Help" tab, user can get help for print barcode.
