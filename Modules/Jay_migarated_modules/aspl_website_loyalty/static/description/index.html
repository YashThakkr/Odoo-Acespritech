Odoo 17 Website Loyalty (Community)

This module provides feature of loyalty in website.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • User can configure loyalty settings from -> Website -> Configuration -> Settings.
  • On the first order of the login user, the reward points is created then the user can use that points afterwards.
  • You can see earned reward points from Website -> Loyalty Points -> Earned Points
  • Now you can redeem your reward on the process checkout page by clicking on redeem reward button.
  • If the configuration is set to Customer Referral you can enter email of referral customer and he will get the referral points on the basis of configuration..
  • You can see your used points by clicking on i button
  • You can see your used points from Website -> Loyalty Points -> redeemed Points
  • If the correct referral email is entered, the points will be added to that partner's points
