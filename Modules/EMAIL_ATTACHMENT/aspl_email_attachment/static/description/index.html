Odoo 17 Email Attachment (Community)

This module allows user to send attachments in email.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Easily send attachment in email.

Available for Enterprise

How to use

  • You can add attachment by clicking on Add attachments link button.
  • Click on Send by Email button , it will add the attachments as per below image.
  • Click on the send button for sending attachments in email.
  • Here you can see the attachments in email
