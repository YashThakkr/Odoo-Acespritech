Odoo 16 Product Low Stock Notification (Enterprise)

Note: This module is tested and working in Odoo Enterprise version. And we are supporting Ubuntu OS and not Windows OS.
Available for Community

Features

  • Update alert quantiy in stockable products.
  • Generate report which show products detail which is less than or equal to product alert quantity.
  • Scheduler which run daily to send email to particular users with product alert report.


How to use

  • Create multiple email templates for product alert quantity. Settings > Email Templates > 'Email : Product Alert Quantity'



  • You can configure the user whom you want to send email for product alert quantity and select the particular template for email.


    • Click on Inventory > Products > Apply Alert Quantity
    • Update product alert quantity based on products, categories and for all. It will give you option to add alert quantity for locations.


    • Once you click on "Apply" button it will update alert quantity in products. Inventory > Products > Product Variants




    • If you checked (Same For All) it will update same alert quantity for all physical locations.



    • Generate report which show products detail which is less than or equal to product alert quantity.









    • Scheduler which run daily to send email to particular users with product alert report. Settings > Technical > Scheduled Actions
