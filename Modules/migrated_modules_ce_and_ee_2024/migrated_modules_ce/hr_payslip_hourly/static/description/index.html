Odoo 16 Employee Pay Hourly (Community)

we can create Employee Payslip based on hours. Here we are going to calculate employee working hours based on attendance.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • First, when we install module. We will get new salary rule for calculating employee wage.


  • After that create new salary structure or into existing salary structure add new salary rule.


  • If rule not present in the salary rules then Select/Create salary rule to existing salary structure.


  • Then create contract for employee and if we want to make employee salary based on hours so make 'Hourly Pay' field checked.


  • Now create employee payslip, after selecting employee name you can see total hours of work will come in 'Total Hours' field. That is based on employee attendance.Now save record and click on compute sheet.
