Available for Enterprise

Features

Dashboard

Building a Dashboard using default Odoo app

Branch

Manage multibranch for multicompany.

Member Profile

Manage customer profile and their regular measurements.

Membership

Manage creation, expiry and renewal of membership.

Gym Services

Manage all the services like yoga, aerobices, etc..

Gym Equipments

Manage gym equipments and its maintenance.

Gym Exercise

Manage all exercise for each service and plan.

Gym Plan

Manage gym plans and schedules for each subscriber.

Nutrition

Manage nutritients from food, nutrition plans and schedules.

Portal Access

Easily access gym and nutrition plan schedules, membership history and renewal on portal.

Reports

Reports for gym and nutrition schedules.

Trainer Profile

Manage trainer for each service and exercise.

Special Offers

Manage special offers for gym plans for selected time period.

How to use

Configurations

  • Go to Fitness Management >> Configuration >> Settings and enable settings as per your requirements.


  • Configure The Branch As Per The Requirement from Setting > User & Company > Branch Menu.


  • After Configure the Branch We can Configure Branch Duration.
  • Fitness > Configuration > Batch Duration you can create a Branch specific schedule and batch duration.


  • Pre Define the Physical Activity.


  • Define Risk Factor Types.


  • Pre Define Member Goal.


  • Define Allergy Types.


  • Defile Symptoms Gym.


  • Defile Medication Type.


  • Create A Member Ship from Fitness > Configuration > Membership


  • Configure Required Information for Membership, Like Fees, Category, Branch, Taxes Etc.


  • Manage Additional Service Which you Provide to Customer.
  • Create A Service from Fitness > Gym > Service


  • Manage Different Plans Along with Different Group of Service.
  • Create A Plans From Fitness > Gym > Plans


  • Configure Food Nutrients.


  • Configure the Unit of Meager to count calorie.


  • Configure Food Items.
  • Fitness > Nutrition > Food > Food Item


  • In Food Items We can Add Description and Configure Nutrients.


  • We can Configure Meal Schedule for Each Member.


  • Dashboard : Able to get All Information from One Screen and Manage All Status As Well.


  • Branch Dashboard: Able to view Branch Related All Information.
