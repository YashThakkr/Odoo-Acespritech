Odoo 17 Hobex Payment Gateway (Community)

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • Hobex Creditcard, Hobex Paypal and Hobex Sofort Payment providers are provided under this module.

  • Hobex Credentials: Take Access Token, Hobex Entity Id and URL put into Hobex Credit Card Payment provider in Odoo, and same for Hobex Sofort Payment provider and Hobex Paypal Payment provider.



  • Company currency and API currency should be same
  • Here need to configure Payment Journal

  • If you want to generate invoice at the time of online payment then you have to configure this settings

  • Go to the website and make order
  • First let's Pay with Hobex Credit Card
  • Select Hobex Credit Card from payment methods.

  • Then you need to enter the card details, Expiry date, CVV and then follow this payment step.

  • User can see confirm order detail here.

  • Now let's Pay with Hobex Paypal
  • Select Hobex Paypal from payment methods.

  • Then you need to follow this payment step



  • User can see confirm order detail here

  • Now let's Pay with Hobex Sofort
  • Select Hobex Sofort from payment methods.

  • Then you need to follow this payment step





  • User can see confirm order detail here

  • Invoicing > Configuration > Payment Transaction inside this menu you can see the Payment Transaction.
  • Inside the Payment Transaction you can see the Sale Order and Invoice Details.

  • Here you can check sale order and invoice in details
