Odoo 16 Direct Attendance CheckIn (Community)

Module allows You to restrict the user/employee to checkin first after login.


Available for Enterprise

How to use

  • You need to give access right to user/employee to direct checkin.


  • After give access right to user/employee, user can see the below screen to checkin
  • Without Check In user can not do anything.
