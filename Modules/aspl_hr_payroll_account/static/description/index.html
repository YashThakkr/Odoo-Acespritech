Odoo 16 Odoo HR Payroll (Community)

HR Payroll Account for Community Version

Features

  • Manage Accounting of Payslip

How to use

  • This Module manage account of salary slip in Odoo. This module helps you managing accounts easily. Into user settings give rights for Analytic Accounting.



  • Select Debit Accout, Credit Account and Tax in Salary Rules.



  • Select Salary Journal in Contract.



  • Salary Journal is set from Employee Contract to Payslip.



  • Salary Journal is set from Employee Contract to Payslip Branch.
