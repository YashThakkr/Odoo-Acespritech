Odoo 16 User RFID Login (Enterprise)

Note: This module is tested and working in Odoo Enterprise edition and tested in community version. And we are supporting Ubuntu OS and not Windows OS.

Available for Community

Features

  • User can Login with RFID Card.

Requirements

  • RFID Device required for this module. Make sure you have connected the RFID Device with the System.

How to use

  • General Settings -> Enable RFID.


  • In the Users form there is field for RFID, Put cursor on that field and scan RFID card with device.


  • User can login by scanning the RFID card from the login page.


  • Scan RFID card With Device.
