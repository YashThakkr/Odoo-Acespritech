Odoo 17 Employee Loan (Enterprise)

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

Features

  • Manage Employee Loan and Integrated with Payroll.
  • Calculate the EMI based on flat and reducing method.
  • Manage Pre-Payments of EMI.
  • Change the Annual Rate of Loan Type and update the rate in loan application.

How to use

  • You can see the available Custom Loan menu.
  • Here we can define different applicant category for loan and also define the documents which required for category.
  • As per below image we can see the configuration for default loan documents in Loan Document menu.
  • Configure accounts for loan operations in Loan setting menu and also you can set installment start day.
  • In Loan Type menu you can create the multiple type of loans and also assign min and max amounts, min and max terms, rate, required documents, also assign the applicant category for loan.
  • In loan application form you upload the required documents and verify the documents.
  • Flat rate calculation.
  • Floating rate calculation.
  • Salary structure with EMI rule.
  • Employee Payslip with EMI.
  • EMI status become PAID after confirmation of payslip.
  • In Pre-Payment menu you can pay Advance payment against Loan.
  • Once advance payment apply it reduce the emi amount.
  • In Change Rate menu you can Calculate the new interest rate.
  • Remaining Unpaid EMI changed after interest rate modified.
  • In Application Form You can print once the loan is disbursed Loan Summary Report.
  • If payslip is Refunded, It recompute salary and amount of EMI.
  • If Payslip is Refunded than Loan Payment State Changed into draft.
