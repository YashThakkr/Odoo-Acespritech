Odoo 17 POS Delete Orders (Community)

Module allows to delete pos orders which are only in paid state.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Delete single/multiple pos orders of different sessions which are in paid state.
  • After delete pos orders it will affect in the payments of pos session.
  • After delete pos orders it will return delivery order as well.
Available for Enterprise

How to use

  • Configure security pin and give allow user for delete paid pos orders.


  • User can select multiple orders of different sessions and delete from list view using select 'Paid Order Delete' option.


  • User can delete single order from for view of pos order using select 'Paid Order Delete' option.


  • After select pos orders user will get pop up where they can enter user pin and process of delete an orders.
