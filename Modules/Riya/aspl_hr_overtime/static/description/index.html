Odoo 17 HR Overtime (Community)

Manages the Overtime for Employee based on Working Schedule.

Payslip is integrated with the Employee Overtime.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • We have provided the feature that lets you make Salary Slip having the overtime calculated as per the Working Schedule of the employee.
  • Working Schedule for the Employee can be selected from here as shown in image.


  • Here is the Employee Overtime Details in Employees -> HR Settings tab. We can set the OT rate for the weekdays , weekends , holidays for perticular employee"


  • If we don't have OT rate set in Employee Form, the OT rate (which is set in Overtime -> Configuration -> Settings) will be considered.
  • Here we can set the OT Time Difference Limit as well.
  • This Working Schedule will be considered if no Working Schedule is selected in Employee Form.
  • If configured manual overtime request it will create manually overtime data otherwise overtime data create with scheduler automatically..


  • Employee can take a look at his/her Overtime Record by clicking the highlighted Smart Button.


  • Employee can see only his/her Overtime Records..


  • HR Manager can either "To be approved" the said Overtime or can Cancel that .
  • Overtime Menu is as shown in image.
  • After To be Approved HR Manager can either Approve the Overtime or can Cancel that.
  • HR Manager can see all the Overtime records.


  • Once the Overtime is Approved, it can't get Cancelled.


  • We need to add one Salary Rule called 'HR Overtime Salary Rule to include the Overtime computation in Payslip'.


  • By computing the sheet, we can have the calculated Overtime in Accounting Information.


  • Salary Computation is as shown in image.


  • Once we Confirm the Payslip, the state for related Ovetime will be changed to 'Paid'.




  • To Print overtime report menu Overtime -> Report -> Overtime.
  • When you click to the Menu you will get this wizard,in this wizard you have to select particular "Start Date","End Date" and "Employee" for Generating Report of Overtime .
  • In wizard Payslip Detail Boolean field. If "Payslip Detail" Boolean True than in report visible payslip reference and status of payslip .
