Odoo 15 Website Gift Voucher (Community)

This module allows to give discount to Customer.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Easy to manage voucher coupon code.
  • Give discount to Customer.
  • Set discount type to fixed or percentage.
  • Discount calculation based on Order total and Product category.
  • Display all vouchers with details in voucher page.
  • Display notification for vouchers based on configuration setting

Available for Enterprise

How to use


  • In Website configuration setting, under the Gift voucher, set the option for enable gift voucher and enable discount for child category and set the option for voucher display.


  • Create Vocucher from here : Website > Gift Voucher > Voucher



  • Home page slider.


  • Display voucher notification.


  • Click on voucher menu and it will displays all available vouchers. When you click on copy button it will copy code to clipboard and you can see description of voucher when you click on view Details.


  • From payment screen you can apply for the voucher code.




  • Generated Sale order and invoice with discounted value of appplied voucher.




  • In Website admin, under the gift voucher tab click on Redeem Voucher to see the redeemed voucher details.
