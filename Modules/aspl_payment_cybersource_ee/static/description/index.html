Odoo 17 Odoo CyberSource Payment Gateway (Enterprise)

CyberSource Features for Enterprise Version

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. And we are supporting Ubuntu OS and not Windows OS.
This module doesn't support 3D secure payment authentication.

Features

  • Ability to transact payments in over 190 countries and territories and fund in over 40 currencies
  • Acceptance of universal cards, regional cards, and local payment types to maximize sales
  • Connections to approximately 100 providers/processors worldwide
  • Unparalleled processing scalability and security

External Dependencies

  • pip3.10 install suds-py3

Available for Community

How to use

  • Website > Configuration > Payment Transaction
  • From the list of payment provider select Cybersource


  • Cybersource Credentials: Take Merchant ID & Cybersource Key and put it into Cybersource Payment Provider in Odoo.


  • Here need to configure Journal


  • If you want to generate invoice at the time of online payment then you have to configure this settings


  • Now let's do payment from website so open website and make order
  • Select Cybersource from payment methods.


  • User can see confirm order detail here


  • Transaction Record has been created in Payment Transaction


  • Here you can check sale order and invoice inside transaction


  • Here you can check sale order and invoice in details
