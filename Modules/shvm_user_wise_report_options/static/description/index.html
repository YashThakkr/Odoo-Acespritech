User Wise Report Printing Options

Live Preview

  • Features
  • Version

  • This module allows to the print or open the report user wise.
  • User can print, open or download the report.

Configuration

  • Users >> Report Options

Report Option: Always Print

  • By selecting this option, user can get the direct print of report.

Report Option: Always Open

  • By selecting this option, reports will direct open in the new browser tab.

Report Option: Always Download

  • By selecting this option, reports will direct download in the system.

Version: 1.0

  • There are four Report Printing Options: Always Ask, Always Print, Always Open, Always Download
  • Always Ask: When user generate any PDF report, It opens the popup for user choice.
  • Inside the popup there are three options: Print, Open, and Download.

Version: 1.1

  • Removed the option, Always Ask from the Report Printing Option.
  • Optimized the code.
