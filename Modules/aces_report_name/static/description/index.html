Odoo 17 Prefix and Suffix in Report Name (Community)

Add Prefix and Suffix to Report Name.

Note: This module is tested and working in Odoo Community edition but not tested in Enterprise edition. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • Activate the developer mode.
  • Go to Settings > Technical > Reporting > Report Configuartion menu.
  • Based on model multiple reports can be added and give Prefix and Suffix in report name.


  • Customized Report Name with Prefix and Suffix.
