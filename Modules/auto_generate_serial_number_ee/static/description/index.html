Odoo 16 Auto Generate Lot/Serial Numbers (Enterprise)

Module allows to generate the Lot/Serial numbers without open wizard for assigning Lot/serial no.

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Generate Lot/Serial numbers for all products using one click.
  • Generate Lot/Serial numbers individually from the list view.

How to use

  • Configure products with the option by lots or by unique serial in tracking option.




  • Create incoming shipment(Receipts) for above two products.


  • For first product we configured by unique serial no. so it has generated unique serial no. for all quantities .


  • For the second product we configured track by lots so it has generated single serial with all quantities.
