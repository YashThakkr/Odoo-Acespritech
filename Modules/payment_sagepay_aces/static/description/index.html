Odoo 17 SagePay Payment Gateway (Community)

We can use SagePay Virtual Payment Gateway for our-2Odoo e-commerce to enhance our business. Customers can pay by redirect to SagePay Virtual Payment checkout page. Redirects back to the original website after successful payment.

Note: This module is tested and working in Odoo Community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

External Dependencies

  • pip3.10 install pycrypto
  • pip3.10 install pycryptodome
Available for Enterprise

How to use

SagePay Payment Provider

  • From the list of payment providers select Sagepay


  • Take Vendor, VPS Protocol and Security key from your SagePay account and put into SagePay Payment Provider in Odoo.


  • Select SagePay from payment methods.


  • For generating invoice after payment
  • Settings --> Website --> Search for "Automatic Invoice"


  • Select your preferred payment method.


  • Fill out the Specified Details if Specified Details is correct then Payment will process and Transaction will happen.


  • Review your order with Transaction detail, Payment Detail and Addresses.


  • Here is the Purchase authentication confirmation.


  • Here is the SagePay payment confirmation.
